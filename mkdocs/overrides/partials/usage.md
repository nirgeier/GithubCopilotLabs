- There are several ways to run the GitHub Copilot Labs.
- Choose the method that works best for you.
  - 📜 From Source (VS Code)
  - 🌐 Using GitHub Codespaces

=== "📜 From Source (VS Code)"

    For those who prefer to run it directly on their machine:

    ```bash
    # Clone the repository
    git clone https://github.com/nirgeier/GithubCopilotLabs.git
    # Change to the Labs directory
    cd GithubCopilotLabs/Labs
    # Start with the Getting Started lab
    cd 001-GettingStarted
    # Follow the instructions in the README of each lab
    cat README.md
    ```
    **Prerequisites:**

    - Visual Studio Code installed
    - GitHub Copilot extension installed and active subscription
    - Git installed

=== "🌐 Using GitHub Codespaces"

    - GitHub Codespaces provides a free, browser-based environment with all necessary tools pre-installed.
    - Click on the `Code` button on the GitHub repository page and select `Open in Codespaces`.

    **Benefits:**

    - No local installation required
    - Pre-configured environment with GitHub Copilot
    - Works on any device with a web browser
    - All tools pre-installed
    - Free tier available
