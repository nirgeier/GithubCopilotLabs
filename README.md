# GitHub Copilot Hands-On Repository

- A collection of Hands-On GitHub Copilot labs.
- Each lab is a standalone lab and does not require completing the previous labs.
- Labs cover the full spectrum of GitHub Copilot features - from basic code completion to advanced Agent Mode.

## Pre-Requirements

- Visual Studio Code (latest version)
- A valid **GitHub Copilot** subscription
- GitHub Copilot extension installed in VS Code
- Basic programming knowledge

| [010 - Agent Mode](Labs/010-AgentMode/)                      | Use Copilot Agent Mode for autonomous multi-step task execution.                 |

| [018 - Code Review](Labs/018-CodeReview/)                    | Use Copilot as a pre-submit reviewer and on GitHub.com to catch issues early.    |
| [019 - Debugging](Labs/019-Debugging/)                       | Diagnose runtime errors, trace stack traces, and fix bugs faster with Copilot.   |
